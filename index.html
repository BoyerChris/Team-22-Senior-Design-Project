What is Check-The-List?


Check-The-List is a cybersecurity hardening solution designed to help its users improve the security of their systems. Within Check-The-List, users will find a variety of tools. Some tools include scripts like vulnerability scanners that perform automated checks in areas commonly prone to compromise, while other tools assist the end user with setting up configurations that allign with best practices in system security.
