Welcome to Check-the-List

Start here!

This guide will walk you through the process of downloading and running a vulnerability scanner from our website. You'll learn how to interact with the initial popup, complete the awareness module, download the scanner, and run it on your system. Follow the steps below to get started.

Step 1: Select your scanner

When you visit the tools page of the website, select the vulnerability scanner you wish the use. You can do this by simply clicking on the button of the scanner you wish to use as shown on the tools page.

Step 2: Complete the awareness module

You will be directed to the awareness module, which provides key information about cybersecurity threats and how the vulnerability scanner works. Read through the material, and then click "Download" when ready.

Step 3: Run the Scanner

After the download completes, a notification will typically appear in the top right corner of your browser. Click on the notification or the Downloads icon. Locate the downloaded file, then run the scanner.

Final Tips

If your operating system requests permission to run the file, select "Yes" or "Allow" to proceed. Follow any additional installation or setup instructions displayed by the scanner. For a complete system check, consider running the scanner as an administrator.


Computer Vulnerability Scanners

Patch Scanner

What is a Patch Scanner?

Our Patch Scanner is a scanner designed to identify missing updates on a Windows system by leveraging built-in Windows features. It scans for security updates, cumulative patches, and third-party software updates, categorizing them for easy identification. This script requires administrative privileges to ensure full access to update information, helping maintain system security and performance.

How it Works

When missing updates are detected, our Patch Scanner generates a detailed report listing all pending patches. It achieves this by leveraging PowerShell's capability to retrieve Windows updates and perform additional tasks using the Get-WindowsUpdate command. If no updates are found, the script confirms that your system is fully up to date. This scanner helps maintain system security by ensuring critical updates are identified and applied promptly.

How to Use

After downloading the Patch Scanner, simply double-click the executable file to run the tool. You will receive a prompt from your computer requesting administrative privileges; please confirm this prompt. If you do not receive the prompt, please right-click the executable and select "Run as administrator." Once launched, the program runs automatically, scanning your system for any missing Windows updates. When the process is complete, you will receive a detailed list of necessary patches, and a file containing this information will be saved for future reference on your C: drive.

Backup Scanner

What is a Backup Scanner?

Our Backup Scanner is a scanner that verifies a system's backup readiness by checking the status of File History and System Restore. It ensures that File History, which provides continuous file backups, is enabled and confirms the availability of System Restore points. If either feature is disabled, the scanner notifies the user and recommends enabling them for enhanced data protection.

How it Works

Baseline Scanner

What is a Baseline Scanner?

Our Baseline Scanner is a security tool designed to identify vulnerabilities that could lead to ransomware attacks, data breaches, or unauthorized access. It strengthens small and medium-sized business (SMB) security by automating critical security checks, supporting compliance with industry standards like NIST, and enabling proactive threat prevention. Regular scans help ensure the ongoing protection of your systems and data.

How it Works

The scanner automates security assessments and exports the findings for easy review. This scanner checks many things, such as whether Remote Desktop Protocol (RDP) is enabled—a common attack vector—retrieves local user accounts and password change dates to identify and enforce password rotation, and reviews firewall rules for misconfigurations such as "Allow All Traffic." These insights help enhance system security by addressing critical vulnerabilities."

How to Use

After downloading the Baseline Scanner, double-click the executable file to launch it. You will receive a prompt requesting administrative privileges—confirm this to proceed. If the prompt does not appear, right-click the executable and select "Run as administrator." The tool will then run automatically, analyzing key security settings. Once the scan is complete, you will receive a detailed report outlining the status of various baseline security configurations, along with a CSV file containing the full results for future reference.

Brute Force Scanner

What is a Brute Force Scanner?

Our Brute Force Scanner is a security tool designed to detect potential brute-force attacks and suspicious network activity. It monitors failed login attempts within the last 30 minutes, identifying repeated failures that could indicate an attempted intrusion. Additionally, it tracks active network connections and flags unknown or excessive connections from specific IP addresses, which may signal a security threat or system overload attempt.

How it Works

This scanner enhances security by identifying unusual login activity and monitoring network threats in real-time. By detecting patterns of repeated failed logins and excessive network connections, it helps users take proactive security measures, such as blocking suspicious IPs or strengthening authentication controls. Early detection of these threats can prevent unauthorized access and protect system integrity.

How to Use

After downloading the Brute Force Scanner, double-click the executable file to launch it. You will receive a prompt requesting administrative privileges—confirm this to proceed. If the prompt does not appear, right-click the executable and select "Run as administrator." The scanner will then run automatically. Once completed, it will provide a detailed report of failed login attempts and suspicious IP addresses, flagging potentially harmful connections for future reference.

Network Adapter Scanner

What is a Network Adapter Scanner?

A Network Adapter Scanner is a security tool designed to analyze your system’s network adapters for their NetBIOS and DNS configurations. It retrieves and displays these settings for each adapter and assesses whether the configurations are classified as “Risky” or “Secure.” By identifying misconfigurations, this tool helps users detect potential vulnerabilities that could expose their system to cyber threats.

How it Works

This scanner is essential for detecting attack vectors caused by improper network adapter settings. It achieves this by leveraging PowerShell's capability to retrieve information from network adapters and the registry using the Get-NetAdapterAdvancedProperty and Get-DnsClientServerAddress commands. Ensuring secure configurations helps reduce the risk of unauthorized access or data exposure through network misconfigurations.

How to Use

After downloading the Network Adapter Scanner, double-click the executable file to launch it. You will receive a prompt requesting administrative privileges—confirm this to proceed. If the prompt does not appear, right-click the executable and select "Run as administrator." The scanner will then run automatically. Once completed, it will generate a detailed report listing all network adapters, highlighting any misconfigurations, and indicating whether their settings are secure or need attention. If a “Risky” configuration is detected, users are advised to take note of the affected adapter and, if possible, disable it until corrective actions can be taken.
