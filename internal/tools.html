Computer Vulnerability Scanners

Patch Scanner

What is a Patch Scanner?

This script is designed to check for missing updates on a Windows system, it does this by utilizing already built-in Windows features. It scans for Windows security updates, cumulative patches, and third-party software updates. If updates are found, they are displayed and categorized respectively, ensuring clear identification of required patches. The script runs only with administrative privileges to ensure proper access to update information


How it Works

Upon detection of missing updates, the script generates a detailed report listing all pending patches. The results are saved in a folder that is created in your C: Drive for reference. If no updates are found, the script confirms that the system is fully updated. The tool is useful for maintaining system security by ensuring critical updates are applied in a timely manner.

Backup Scanner

What is a Backup Scanner?

This script checks the system's backup readiness by verifying the status of File History and System Restore. It ensures that File History, which provides continuous file backups, is enabled, and checks for the availability of System Restore points. If either feature is disabled, the script notifies the user and suggests enabling them for added protection.


How it Works

The results are displayed and then stored in a newly created folder on your C drive. This allows users to review their backup settings and take corrective action if needed. The script must be run as an administrator to access system settings properly. By using this tool, users can quickly determine if their system has adequate recovery options in place, reducing the risk of data loss in case of system failure

Password Scanner

What is a Password Scanner?

This script evaluates the local security policies related to password management on a Windows system. It retrieves and displays key password security settings, including minimum password length, maximum password age, password complexity requirements, and account lockout thresholds.


How it Works

The scanner analyzes these settings and provides recommendations if weak policies are detected, such as enabling password complexity, setting a stronger minimum length, or enforcing password expiration. The results are displayed in a structured format, with security warnings highlighted in color for better visibility. Using these outputs, you can quickly assess and improve their system's password security policies, reducing vulnerabilities related to weak authentication settings.

Baseline Scanner

What is a Baseline Scanner?

This scanner helps prevent ransomware, breaches, and unauthorized access by identifying security gaps. It strengthens SMB security by automating checks, supports compliance with standards like NIST, and enables proactive threat prevention. By scanning regularly, you can ensure the ongoing protection of your assets


How it Works

This scanner automates security checks and exports findings to a CSV file. It checks if Remote Desktop Protocol (RDP) is enabled, a common attack vector, and retrieves local user accounts and password change dates to identify stale accounts and enforces rotation, and reviews firewall rules for misconfigurations like "Allow All Traffic."

Network Vulnerability Scanners

Port Scanner

What is a Port Scanner?

This script scans your system searching for ports that have been left open, such as FTP, SSH, HTTP, and RDP. It systematically attempts to establish a connection with each port and reports whether it is open or closed. If a port is open, the script confirms connectivity and then closes the connection.


How it Works

The tool is useful for identifying accessible services on a system, which can help with security assessments and network troubleshooting. The scan results are displayed in real-time, showing the status of each port.

Brute Force Scanner

What is a Brute Force Scanner?

This script performs two key security checks. It scans for failed login attempts in the last 30 minutes, identifying potential brute-force attacks by tracking repeated failures per user. It also monitors active network connections, detecting suspicious or unknown IP addresses. The script flags IPs with excessive connections, which could indicate an intrusion or system overload attempt.


How it Works

This tool helps detect attacks early by spotting unusual login activity and monitoring network threats. It improves situational awareness, allowing proactive actions like blocking suspicious IPs before an attack succeeds.

Network Adapter Scanner

What is a Network Adapter Scanner?

This script scans your systems network adapters for their NetBios and DNS configurations. It retrieves and displays the aforementioned configurations for each adapter your system possesses, determining along the way if the configuration found was “Risky” or “Secure”.


How it Works

This scanner is useful for finding attack vectors resulting from poor network adapter configuration. Upon detection of a “Risky” setting, the end user should take note of the adapter affected by the misconfiguration, then if possible, disable the adapter until steps are taken to rectify the affected DNS or NetBios configuration.
